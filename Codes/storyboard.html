Storyboard and Design Plan

This page presents the design plan for my portfolio website. The design uses a soft black and brown theme to create a calm, comfortable, and professional appearance.

Mobile Layout Design

The mobile layout is designed as a single-column structure. The logo and navigation appear first, followed by the page heading, image, content sections, cards, and footer.

Save your mobile storyboard image as mobile-wireframe.jpg inside the Assets folder, then uncomment the image line below.

Desktop Layout Design

The desktop layout uses wider sections and grid-based cards. The hero section places text on one side and the profile image on the other side. Other content uses two-column and three-column layouts where appropriate.

Save your desktop storyboard image as desktop-wireframe.jpg inside the Assets folder, then uncomment the image line below.
